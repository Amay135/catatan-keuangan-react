# Catatan Keuangan React

Aplikasi pencatatan keuangan harian berbasis React dan Bootstrap untuk membantu pengguna mengelola pemasukan dan pengeluaran dengan mudah.

---

## Fitur Utama

- Menambahkan transaksi baru
- Edit data transaksi
- Hapus transaksi
- Kategori pemasukan & pengeluaran
- Perhitungan saldo otomatis
- Ringkasan keuangan bulanan
- Format mata uang Rupiah (IDR)
- Tampilan responsif menggunakan Bootstrap

---

## Teknologi yang Digunakan

- React.js
- Bootstrap 5
- JavaScript (ES6)
- Vite
- CSS

---
